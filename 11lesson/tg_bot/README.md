```
config.py

Этот файл содержит конфигурационные настройки для бота, 
такие как токен API и настройки логирования. 
Здесь также инициализируются объекты Bot и Dispatcher, 
которые используются для взаимодействия 
с Telegram API и обработки сообщений.
```

```
database.py

Этот файл содержит функции для работы с базой данных SQLite.
Здесь определены функции для инициализации базы данных,
сохранения результатов пользователей и получения статистики.
```

```
main.py

Этот файл является точкой входа для запуска бота. 
Здесь инициализируется база данных и запускается 
цикл опроса для обработки входящих сообщений.

А так же обработчики сообщений и колбэков для бота.
Здесь определены функции для начала квиза, 
обработки ответов пользователей, 
отображения результатов и вывода статистики.
```


```
quiz_data.py

Этот файл содержит список вопросов и ответов для квиза. 
Каждый вопрос представлен в виде словаря 
с ключами question, options и correct_answer.
```


```
utils.py

Этот файл содержит вспомогательные функции, такие как 
создание клавиатуры для ответов на вопросы. 
Эти функции помогают упростить основной код бота.
```
